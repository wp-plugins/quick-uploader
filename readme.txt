== Description ==



Sometimes you need to Upload image or other files & copy link to media file only. In default system of wordpress you'll have to upload the file, go to media page & copy the url. So lengthy .. right ?

Here's the Quick Uploader Comes in! You can now upload and get link in same page within seconds.

This plugin adds a new menu named Quick Uploader under Media menu in wordpres admin panel. From that menu you can click on upload button, select file & get link to uploaded file instantly without going to any other page. Pretty simple!

<h3>Features :</h3>

* Simplify Upload Process
* Upload and get link to media file instantly
* Lightwight Plugin

== Installation ==

Installing this plugin is pretty simple like all other wordpress plugins. Here are 2 ways to install :

1. Manual Install

* Download the zipped plugin from wordpress repository
* Extract the zipped folder 
* Upload `quick-uploader` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Go To 'Media -> Quick Uploader' menu to configure options

2. Automatic Install

* Go to Plugins-> Add New from your wordpress Dashboard
* Search Quick Uploader 
* From the results click on 'install now'
* Activate the plugin through the 'Plugins' menu in WordPress
* Go To 'Media -> Quick Uploader' menu to configure options
